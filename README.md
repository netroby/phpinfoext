php info extensive
==================

Introduction
-------------

This tools can help you view php info details.

1. PHP functions
2. Loaded extensions
3. Zend OPCache
4. phpinfo
5. memcached
6. redis

Installation
------------
````
git clone https://github.com/ancientrock/phpinfoext.git
````

Access url: http://domain.com/phpinfoext/t.php

Replace the url to your real path.
